# 游戏平台页面切图素材

- `page_visible_without_black.png`：去掉底部黑屏后的完整可见页面，适合做 Codex 视觉对照。
- `header_blue_area.png`、`main_white_panel_top.png`：大区域背景切片。
- `*_full.png`：完整组件切片，可直接作为页面元素。
- `icon_*` / `label_*`：卡片插画和标题分离切片。
- `assets_coordinates.json`：每个素材在原图中的像素坐标，方便在 CSS 中按比例定位。

说明：原图下半部分是纯黑遮挡，因此第二排游戏卡片只保留了可见的顶部 20px 片段。
